# CLI Scope

## Command surface

- `aibridge init`
- `aibridge status`
- `aibridge task add|assign|in-progress|done`
- `aibridge log`
- `aibridge handoff`
- `aibridge decision`
- `aibridge convention add`
- `aibridge sync`

## Notes

CLI writes protocol-compliant files into `.aibridge/` and triggers context regeneration.
